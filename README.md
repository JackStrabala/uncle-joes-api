# Uncle Joe's API

This repository contains the backend API for the Uncle Joe’s Coffee Company pilot web application.

## Project Overview
Uncle Joe’s Coffee Company is building an internal pilot to modernize its digital presence. This application uses historical company data to test a customer-facing platform before a wider rollout across nearly 500 locations.

The backend is built with FastAPI and connects to Google BigQuery. It provides REST API endpoints that allow the frontend application to retrieve menu data, store location information, Coffee Club member details, order history, and loyalty points balances.

## Main Features
- Retrieve menu items from BigQuery
- Retrieve Uncle Joe’s store location data
- Support Coffee Club member login
- Return customer order history
- Calculate loyalty points from historical order data

## Purpose of This Repository
This repository is responsible for the server-side logic and data access layer of the project. It handles requests from the frontend and returns the appropriate data from the Uncle Joe’s dataset.
